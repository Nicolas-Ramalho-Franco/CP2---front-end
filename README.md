Cantio — Sabor com Alma

A Cantio é uma landing page moderna e responsiva desenvolvida para uma lanchonete artesanal fictícia. 
O foco do projeto é transmitir acolhimento e qualidade através de um design limpo, tipografia elegante e uma experiência de usuário intuitiva.

Sobre o Projeto
O nome Cantio remete ao cuidado e à tradição. 
A página foi construída para apresentar a história da marca, o cardápio diversificado (com foco em caldos, lanches e sobremesas) e os diferenciais que tornam o estabelecimento único.

Este projeto faz parte de uma entrega acadêmica da FIAP, unindo conceitos de Web Development, UI/UX e estruturação semântica de HTML5.

Tecnologias Utilizadas
O projeto foi construído utilizando as melhores práticas de desenvolvimento Front-End:

HTML5: Estruturação semântica de toda a página.

CSS3: Estilização personalizada, layouts em Flexbox/Grid e design responsivo.

Google Fonts: Utilização das fontes Playfair Display (elegância) e Lato (legibilidade).

Font Awesome: Biblioteca de ícones vetoriais para uma interface mais visual.
